# The World Forgot Us

Master Design Bible v0.1

## 1. High Concept

The World Forgot Us is a 2D open-world post-apocalyptic survival adventure where the player explores abandoned ruins, loots supplies, fights to survive, rebuilds a personal base, and recovers memories from a world that is literally forgetting itself.

The world did not only collapse. It began to lose meaning. Names vanish from signs. Photos lose faces. Places remain standing, but nobody remembers what they were for. Radios repeat half-sentences from people who are gone. Whole streets exist on maps one day and disappear the next.

The player survives by gathering food, water, scrap, medicine, weapons, and rare technology. But the deeper progression comes from recovering echoes: fragments of memory attached to rooms, objects, voices, bodies, machines, homes, roads, songs, and ruined places.

The core promise:

> Loot the wasteland. Rebuild your base. Recover what the world forgot.

## 2. One-Sentence Pitch

A 2D open-world survival game where every ruin has a present and a past, and the player survives by looting supplies while rebuilding the lost memory of the world.

## 3. Genre and Camera

Genre:

- 2D open-world survival adventure
- Exploration, looting, combat, base building, story discovery
- Light RPG progression without becoming a huge stat spreadsheet

Recommended camera:

- 2D top-down or 3/4 top-down view
- This supports open-world movement, interiors, looting, combat, base building, and environmental storytelling better than pure side-scroller

Alternative:

- 2D side-view works well for atmosphere and platforming, but makes base building and open-world layouts harder

Best direction:

- Top-down with strong lighting, shadows, fog, rain, ash, dust, and screen-space effects

## 4. Design Pillars

| Pillar | Meaning |
|---|---|
| Survival With Meaning | The player needs supplies, but the emotional drive is recovering lost people, places, and truth. |
| Open Freedom, Soft Guidance | The player can roam, loot, build, and ignore the main thread, but signals, rumours, landmarks, and base upgrades gently guide them. |
| Memory Is A Resource | Memories unlock rooms, crafting, base upgrades, map knowledge, NPC trust, endings, and combat options. |
| Every Place Has Two Layers | Each ruin has its present-day danger and its hidden echo of what happened before or during the collapse. |
| The Base Becomes A Living Archive | The base is home, workshop, safehouse, memorial, museum, and proof that the world can be understood again. |
| Scrappy Combat | Combat should feel tense, rough, and improvised. The player survives with repaired tools, handmade weapons, traps, and scanner tricks. |
| Custom Identity | Character, clothing, backpack, weapons, scanner, room, base layout, and playstyle should all feel personal. |

## 5. What Makes It Unique

The game is not just "post-apocalyptic looting with base building." The unique identity is memory recovery.

Most wasteland games ask:

> Can you survive after the world ends?

This game asks:

> If the world forgets everything, what is worth remembering?

The player does not only open containers for random junk. They investigate places. A petrol station might contain fuel, a weapon part, a hidden basement, an old staff badge, a fight, a recorded argument, a child's drawing, a missing road name, and a memory that unlocks a new route.

Loot can be:

- useful material
- emotional evidence
- a clue
- a trade item
- a base decoration
- a faction bargaining chip
- a crafting unlock
- a piece of someone else's life

## 6. Core Gameplay Loop

1. Wake at the base.
2. Check weather, supplies, radio signals, NPC requests, and unresolved echoes.
3. Choose gear: weapon, scanner module, backpack, armour, food, medicine, tools.
4. Pick a destination or wander freely.
5. Explore the wasteland.
6. Enter ruined buildings, vehicles, tunnels, towers, homes, shops, shelters, and bunkers.
7. Loot supplies and meaningful objects.
8. Fight, sneak around, trap, distract, or avoid enemies.
9. Use the scanner to recover echoes and reveal hidden layers.
10. Return to base before storms, darkness, injury, or over-encumbrance become too risky.
11. Build, craft, upgrade, decorate, talk to NPCs, archive memories, and choose what to preserve.
12. Unlock new zones, story leads, tools, and consequences.

The loop should create "one more run" motivation while still feeling story-rich.

## 7. The Memory System

### 7.1 Memory Echoes

Memory echoes are fragments of the past attached to objects, rooms, people, machines, or places.

Examples:

- a voice line from the last day before evacuation
- a flickering silhouette replaying a final action
- a missing name restored to a sign
- a photo regaining faces
- a room briefly appearing as it used to be
- a radio playing a half-buried message
- a map marker reappearing
- an item revealing its owner and purpose

### 7.2 Echo Layers

Each important location has two layers:

| Layer | Description |
|---|---|
| Present Layer | The abandoned, dangerous, lootable location as it exists now. |
| Echo Layer | A partial memory of what the place was, what happened there, or what was lost. |

The player can scan certain objects, tune frequencies, restore power, place items back where they belong, or solve small context clues to recover the echo.

### 7.3 Memory Quality

Not all memories are complete.

| Quality | Meaning |
|---|---|
| Static | Only a sound, image, or broken phrase. |
| Fragment | A short clue with some context. |
| Clear Echo | A readable event, voice, room state, or personal memory. |
| Restored Memory | A full location or person memory that unlocks gameplay or story consequences. |

### 7.4 Memory As Progression

Recovered memories can unlock:

- map areas
- hidden entrances
- recipes and blueprints
- NPC trust
- faction secrets
- base upgrades
- unique item uses
- new dialogue
- enemy weaknesses
- alternative endings
- restored names and places in the world

### 7.5 Memory As Choice

Some memories can be used in different ways.

Example item: Old Wedding Ring

| Use | Result |
|---|---|
| Scrap it | Gain rare metal for crafting. |
| Archive it | Add a restored couple memory to the base memorial. |
| Give it to an NPC | Complete a personal quest. |
| Trade it | Gain faction favour or supplies. |
| Return it to a house | Unlock a hidden echo and secret stash. |

This makes looting feel less disposable.

## 8. World Lore

### 8.1 Before The Collapse

Before the world fell, governments and corporations built a global emergency communication network called The Choir Grid.

The Choir Grid connected:

- emergency broadcasts
- hospitals
- weather stations
- public transport
- military warnings
- power infrastructure
- city cameras
- schools
- radio towers
- satellites
- home devices
- public screens

It was meant to keep society alive during disasters by making every system communicate instantly.

Then came The First Tone.

### 8.2 The First Tone

One night, every connected speaker, phone, radio, television, siren, traffic light, public screen, and emergency system broadcast the same impossible signal.

People described it differently:

- a choir
- a scream
- a machine praying
- the voice of someone they had lost
- complete silence that somehow made sound

After The First Tone:

- power grids failed
- roads jammed permanently
- hospitals lost records
- evacuation routes contradicted each other
- maps became unreliable
- names disappeared from databases
- weather turned strange
- some people became obsessed with listening
- some places began replaying moments from the past
- whole communities vanished from official records

The disaster became known as The Forgetting.

### 8.3 The Forgetting

The Forgetting is not simple amnesia. It affects the world itself.

It erases connection between things:

- a house remains, but nobody remembers the family
- a road remains, but maps forget where it leads
- a person remains, but others forget their name
- a machine remains, but its purpose becomes corrupted
- a song remains, but no one remembers who sang it

The more forgotten something becomes, the more unstable it is.

Forgotten places become echo sites. Forgotten people can become hollow. Forgotten machines may repeat old commands forever.

### 8.4 The Player's Role

The player is a Runner: someone who travels between unstable zones to gather supplies, information, and lost things.

The player owns or discovers a rare scanner called a Mnemoscope. It can detect memory echoes and tune into forgotten signals.

At first, the player uses it to find supplies. Over time, they realise it can rebuild pieces of the world.

## 9. Main Character

### 9.1 Starting Concept

The player character should be somewhat defined but customisable.

They are not a blank superhero. They are a young survivor from the edge of the dead zones, used to scavenging but not yet important. The story begins when they find an object or signal that should not exist: a memory with their own name in it.

Possible starting backstory:

- They were raised in a small survivor camp.
- They make supply runs to keep people alive.
- Their parent, sibling, friend, or mentor disappeared into a forgotten zone.
- They discover that records of their own past have started vanishing.
- Their base begins as a temporary shelter, then becomes the archive of the recovered world.

## 11. Open World Structure

The world should be handmade, not randomly generated, but with variable loot, enemy positions, weather, and events.

This gives the player freedom without making the world feel empty.

### 11.1 Starting Region

The first region should include:

- The Railhome Archive
- a broken road
- a ruined petrol station
- a few abandoned houses
- a small shop
- a blocked tunnel
- a radio tower visible in the distance
- a dangerous area the player can see but not safely enter yet

### 11.2 Major Districts

| District | Vibe | Gameplay Purpose |
|---|---|---|
| Rustway | Broken motorway, crashed buses, old cars, service stations | Early scavenging, fuel, road ambushes. |
| Hollow Estate | Abandoned homes and flats | Personal memories, medicine, food, emotional lore. |
| Glassmarket | Dead shopping centre | Clothing, tools, factions, rare loot, interior exploration. |
| Blackwater Underpass | Flooded tunnels and maintenance routes | Shortcuts, danger, hidden infrastructure. |
| The Broadcast Fields | Radio towers, pylons, signal dishes | Main story, scanner upgrades, faction conflict. |
| Ashmere School | School ruins and sports halls | strong emotional lore, early major story reveal. |
| St. Vey's Hospital | Hospital and emergency tents | medical supplies, hard enemies, collapse records. |
| The Red Zone | unstable wasteland, storms, corrupted echoes | high risk, rare rewards, late game gear. |
| The Silent City | ruined centre, empty high-rises, dead screens | endgame truth and final decisions. |

### 11.3 Soft Guidance Tools

The game should guide without forcing.

Use:

- visible landmarks
- distant radio towers
- scanner static
- NPC rumours
- old maps
- found notes
- restored road signs
- base radio messages
- storm warnings
- locked doors needing real tools
- dangerous enemies that suggest "come back later"
- environmental lighting pointing at routes

Avoid:

- constant quest arrows
- massive UI clutter
- invisible walls everywhere
- long tutorial popups

## 15. Enemies

| Enemy | Description | Gameplay |
|---|---|---|
| Hollow | Forgotten human-like survivor, identity almost erased | Slow but unsettling, drawn to sound and echoes. |
| Scrap Hound | Machine-animal made from old patrol tech | Fast, hunts by noise, weak to EMP. |
| Static Wraith | Echo entity only visible when scanned | Forces scanner use and movement. |
| Ash Strider | Tall figure seen during storms | Avoid early, fight later with upgraded gear. |
| Relay Drone | Broken maintenance/security drone | Patrols, shoots, can be hacked or disabled. |
| Scavenger Raider | Human enemy seeking supplies | Uses weapons, traps, retreats, can belong to factions. |
| Choir-Touched | Person obsessed with the signal | Erratic movement, attacks with sound bursts. |
| Memory Husk | A person or creature guarding a restored echo | Puzzle-combat enemy tied to specific lore. |

Enemies should not all be mindless. Some should be tragic. Some should be avoidable. Some should become understandable after the player restores the right memory.

## 16. Factions

| Faction | Belief | Role |
|---|---|---|
| The Relay Union | Communication must be restored under strict control | Orderly, useful, dangerous if trusted too much. |

| Salvage Houses | Whoever controls supplies controls survival | Traders, rivals, contracts, black markets. |

Faction reputation should affect:

- trade prices
- access to zones
- NPC trust

## 18. Story Structure

### Act 1: The Name That Should Not Exist

The player finds a memory echo containing their own name or a family name in a place they have never visited.

Goals:

- learn scanner use
- establish base
- discover The Forgetting
- meet first NPCs
- restore first memory
- reach the first radio tower

### Act 2: The World Under The World

The player discovers that The Forgetting is connected to the Choir Grid and that factions are fighting over signal sites.

Goals:

- expand base
- choose allies
- recover district memories
- unlock deeper zones
- discover Old Authority cover-ups
- learn that recovered memories can change current reality in small ways

### Act 3: The Silent City

The player reaches the origin region of The First Tone and learns the truth is not simple.

Possible truths:

- The Choir Grid tried to preserve humanity by storing memory, but broke reality.
- The government weaponised memory and lost control.
- The signal is an alien or unknown intelligence using human systems.
- The world is not forgetting by accident. It is being edited.

The exact answer can stay partly mysterious, but the player should get emotionally satisfying endings.

## 19. Endings

Endings should depend on what the player preserves, destroys, restores, and who they side with.

| Ending | Condition | Result |
|---|---|---|
| Archive Ending | Restore many memories and protect the base archive | The world remains broken, but memory survives and communities begin reconnecting. |
| Silence Ending | Side with The Quiet or destroy the core signal | Echoes stop, danger falls, but many memories are lost forever. |
| Relay Ending | Restore the network through The Relay Union | Order returns, but under surveillance and control. |
| Choir Ending | Merge with or release the signal | Strange, ambiguous ending where memory becomes something beyond human control. |
| Survival Ending | Ignore most main story and focus on base survival | Smaller but valid ending: your people live, but the wider truth remains buried. |

## 27. Claude CLI Build Plan

Recommended engine:

- Godot 4

Reason:

- strong 2D tooling
- free and open source
- lightweight for the laptop
- good for top-down games
- easy project structure
- works well with GDScript

1. Create Godot project structure.
2. Add player movement and camera.
3. Add interaction system.
4. Add inventory data model.
5. Add lootable containers.
6. Add base scene and return-to-base loop.
7. Add one enemy and simple combat.
8. Add scanner pulse.
9. Add memory echo object.
10. Add base upgrade station.
11. Add save/load.
12. Add demo district.
13. Add UI polish.
14. Add sound placeholders.

## 28. Risks and Scope Control

Biggest risks:

- too much lore before playable systems
- too many item types
- base building becoming too complex
- open world becoming empty
- too many generated assets with inconsistent style
- asking Claude to build the whole game in one prompt

Rules:

- build a 10-minute demo first
- every feature must support the core loop
- every major mechanic must connect to memory, survival, or base growth
- keep combat simple until exploration and looting feel good
- use placeholder art until gameplay works
- lock art style before generating lots of assets
- never add a faction, enemy, or item without gameplay purpose

## 29. Current North Star

The World Forgot Us should feel like walking through the remains of a world that almost remembers itself.

The player should feel:

- curious when entering ruins
- tense when supplies run low
- clever when using the scanner
- attached to recovered objects
- proud when the base improves
- uneasy when memories contradict each other
- powerful when they survive dangerous runs
- emotional when names, places, and people are restored

The game stands out when looting, building, combat, and story all point to the same idea:

> In this world, survival is not enough. Someone has to remember.
