# Changelog

## 1.0.0 - 2026-07-11

### Complete campaign

- Expanded the original Rustway slice into Ashmere Verge, Broadcast Fields, and Choir Core.
- Added the Ellie / Mara story payoff, five recoverable echoes, staged dialogue, three relay objectives, and Archive / Silence / Choir endings.
- Added Static Wraith and Relay Husk enemy classes, two shielded boss encounters, dodge, and Memory Burst.
- Added the Archive, ending/credits screen, complete controls, explicit story progression, and save schema v2.

### Presentation

- Added normal-mapped real-time 2D lighting, shadow-casting occluders, dynamic player/scanner/story lights, and chapter palettes.
- Added the memory-lantern chiaroscuro screen grade and expanded procedural sound design.
- Added a custom accessible Web shell with launch, loading, error, save-storage, and fullscreen UX.

### Delivery

- Added a single-threaded Godot Web export preset.
- Added deterministic end-to-end campaign smoke testing.
- Added automated GitHub Pages build and deployment.
