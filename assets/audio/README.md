# Original soundtrack

These tracks were composed and rendered specifically for *The World Forgot Us*:

- `memory-in-the-rain.ogg` / `.mp3` — 84-second seamless exploration ambience
- `receiver-veil.ogg` / `.mp3` — 60-second seamless receiver / high-dread ambience

The Ogg Opus versions are preferred where supported; MP3 fallbacks keep the score available in browsers without Ogg Opus playback. Both pieces are original, sample-free works with no third-party music, loops, performances, or generated voice material.
